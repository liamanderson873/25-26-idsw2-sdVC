# RUP — Jorgestor

Artefactos de la metodología RUP aplicada al sistema Jorgestor. Organizados en tres disciplinas: requisitos (casos de uso), análisis y diseño.

## Disciplinas

|||
|-|-|
[Casos de uso](/RUP/00-casos-uso/01-actores-casos-uso/)|Diagramas de actores (Docente y Administrador Institucional) y diagramas de contexto de navegación.
[Análisis](/RUP/01-analisis/README.md)|43 casos de uso con análisis BCE (Boundary-Control-Entity): colaboración y secuencia de análisis, independientes de tecnología.
[Diseño](/RUP/02-diseño/README.md)|43 casos de uso con secuencia de diseño a nivel de implementación (Spring Boot + React + PostgreSQL) y 4 diagramas arquitectónicos.
[Evolución](/RUP/EVOLUCION_DISENO.md)|Comparativa Baseline vs. As-Built: modelo del dominio y ciclo de vida del examen.
